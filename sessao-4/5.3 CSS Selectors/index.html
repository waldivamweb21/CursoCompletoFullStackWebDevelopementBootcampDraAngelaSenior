Seletores CSS

Aplicando CSS a diferentes partes do HTML

1. O seletor de elementos direciona elementos com base no nome da tag HTML.

  1. Os seletores de classe direcionam elementos com base no valor do atributo class.
  2. Os seletores de ID direcionam elementos com base no valor do atributo id.
  3. Os seletores de atributos direcionam elementos com base em seus atributos e valores.
  4. O seletor universal tem como alvo todos os elementos.
